# Gravitational Bias

Story:
This image frames uncertainty under pressure. Random motion is always present, but an off-center attractor gradually pulls the system into a shared region and leaves a visible memory of convergence.

Method:
Particles begin across the canvas and evolve under a linear attractive force plus diffusion. The result is an Ornstein-Uhlenbeck style process rendered as faint converging trails and a dense terminal cluster.

Parameters:
- Seed: 42
- Particles: 340
- Steps: 720
- Diffusion coefficient D: 0.0022
- Time step dt: 1.0
- Attraction strength alpha: 0.032
- Attractor: (0.24, -0.18)
