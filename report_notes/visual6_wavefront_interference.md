# Wavefront Interference

Story:
Several signal origins pulse across the same space and overwrite one another. The resulting image reads like a frozen acoustic map where coherence and conflict occupy the same surface.

Method:
The scalar field is built from multiple damped radial sine waves emitted from different sources. Glow is rendered from the field magnitude and contour layers reveal the interference bands created by phase differences.

Parameters:
- Seed: 45
- Resolution: 6000 x 6000
- Sources: 3
- Frequencies: [28.0, 24.0, 31.0]
- Amplitudes: [1.0, 0.86, 0.74]
- Decays: [1.30, 1.12, 1.24]
